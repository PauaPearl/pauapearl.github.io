Technical Information: This map highlights the recreational reserve area known as Mount Victoria, in Devonport, Auckland, New Zealand. The map was created using GeoJSON, Leaflet, Google Earth, OpenStreetMap, and hosted on GitHub. Some scripting was done in both Notepad++ and Dreamweaver. Leaflet was used to provide the base map layer. Leaflet is an open-source Javascript library for interactive maps. I used Google Earth to find the latitude and longitude coordinates, as well as draw the polygon. The polygon was then exported as a .kml file into GeoJSON, so it could be converted into a "Feature" and added to the map in Javascript. Stylistic changes were made to the polygon. I added a placemark and popup text. The map was then embedded into a html file. The html document was then uploaded to GitHib for hosting.
